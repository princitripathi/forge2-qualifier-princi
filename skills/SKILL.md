\# Kanban Board Skill



\## Trigger

User wants to create or manage a Kanban board with boards, lists, and cards.



\## Steps

1\. Create a board with a name

2\. Add lists to the board (e.g. To Do, In Progress, Done)

3\. Add cards to lists with title, description, due date, tags, member

4\. Move cards between lists by dragging

5\. Assign members and tags to cards



\## Output

A fully functional Kanban board saved via Laravel API or local state.
